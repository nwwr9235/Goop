# 🤖 ArabBot — بوت المجموعات العربية الشامل

بوت تيليغرام متكامل مبني بـ Python يجمع بين إدارة المجموعات والحماية والردود التلقائية ونظام الترحيب ومشغل الموسيقى.

---

## ✨ المميزات

| الميزة | الوصف |
|--------|-------|
| 👮 **الإدارة** | رفع وتنزيل المشرفين، حظر، كتم، طرد |
| ⚠️ **الإنذارات** | نظام إنذار تلقائي مع إجراء عند الحد الأقصى |
| 🛡 **الحماية** | قفل الروابط، الصور، الفيديو، الملفات، البوتات، السبام |
| 👋 **الترحيب** | رسالة ترحيب قابلة للتخصيص مع متغيرات |
| 💬 **الردود التلقائية** | ردود مخصصة على كلمات محددة |
| 🎵 **الموسيقى** | تشغيل يوتيوب في المحادثات الصوتية مع قائمة انتظار |
| ℹ️ **المعلومات** | معرّفات المستخدمين ومعلومات المجموعة |
| 📡 **الإذاعة** | إرسال رسائل لجميع المجموعات (للمالك فقط) |

---

## 📦 متطلبات التشغيل

- Python 3.11+
- MongoDB
- FFmpeg
- حساب تيليغرام مطوّر (API_ID + API_HASH)
- توكن البوت من [@BotFather](https://t.me/BotFather)

---

## 🚀 طريقة التثبيت

### 1. استنساخ المشروع
```bash
git clone https://github.com/yourrepo/arabbot.git
cd arabbot
```

### 2. إنشاء ملف `.env`
```bash
cp .env.example .env
nano .env
```

أملأ البيانات التالية:
```
BOT_TOKEN=توكن_البوت
API_ID=رقم_الـAPI
API_HASH=هاش_الـAPI
OWNER_ID=رقم_معرّفك
MONGO_URI=mongodb://localhost:27017
```

### 3. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 4. تشغيل البوت
```bash
python main.py
```

## 🐳 التشغيل بـ Docker

cp .env.example .env

## 📋 جميع الأوامر

### 👮 الإدارة
```
رفع @user          — ترقية إلى مشرف
تنزيل @user        — إزالة الإشراف
حظر @user          — حظر مستخدم
الغاء الحظر @user  — رفع الحظر
كتم @user          — كتم مستخدم
الغاء الكتم @user  — إلغاء الكتم
طرد @user          — طرد من المجموعة
عرض الادمنية       — قائمة المشرفين
```

### ⚠️ الإنذارات
```
انذار @user <سبب>      — إنذار مستخدم
عرض الانذارات @user    — عرض الإنذارات
مسح الانذارات @user    — مسح الإنذارات
```

### 🛡 الحماية (القفل/الفتح)
```
قفل الروابط / فتح الروابط
قفل المعرفات / فتح المعرفات
قفل البوتات / فتح البوتات
قفل الصور / فتح الصور
قفل الفيديو / فتح الفيديو
قفل المتحركة / فتح المتحركة
قفل الملفات / فتح الملفات
قفل التكرار / فتح التكرار
قفل السبام / فتح السبام
```

### 👋 الترحيب
```
تفعيل الترحيب
تعطيل الترحيب
تعيين رسالة الترحيب <الرسالة>
  المتغيرات: {user} {group}
```

### 💬 الردود التلقائية
```
اضافة رد مرحبا = اهلا بك!
حذف رد مرحبا
عرض الردود
```

### 🎵 الموسيقى
```
تشغيل <اسم الأغنية أو رابط يوتيوب>
تخطي
ايقاف
ايقاف مؤقت
استئناف
قائمة التشغيل
حذف من القائمة <رقم>
مغادرة
```

### ℹ️ المعلومات
```
ايدي
ايدي @user
معلوماتي
معلومات @user
معلومات المجموعة
```

### 🔧 المالك
```
اعادة تشغيل
حالة البوت
سرعة البوت
اذاعة <الرسالة>
```

## ⚙️ الإعداد المتقدم

### نظام الإنذارات
في ملف `.env`:
```
MAX_WARNINGS=3          # عدد الإنذارات قبل الإجراء
WARN_ACTION=mute        # mute أو ban
```

### قناة السجلات (اختياري)
```
LOG_CHANNEL=-1001234567890
```
